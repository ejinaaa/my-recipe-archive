# 빌드 및 에러 수정

프로젝트를 빌드하고 에러를 자동으로 수정합니다.

## 워크플로우

### 1단계: 빌드 실행

```bash
pnpm build
```

### 2단계: 에러 분석 및 수정

에러 발생 시:
1. 에러 메시지 파싱
2. 해당 파일로 이동하여 수정
3. 수정이 어려운 경우 사용자에게 확인

### 3단계: 재빌드

에러 수정 후 다시 빌드 실행

### 4단계: 결과 보고

```
## 빌드 결과

### 수정된 에러
1. [파일:라인] - [에러] → [수정 내용]

### 최종 상태
✅ 빌드 성공 / ❌ 빌드 실패
```

## 반복

빌드 성공까지 2~4단계 반복
