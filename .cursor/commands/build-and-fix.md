# 빌드 및 에러 자동 수정

pnpm을 사용하여 빌드 스크립트를 실행해줘.

## 요구 사항

1. **에러 분석**: 빌드 중 TypeScript 에러나 Lint 에러가 발생하면, 해당 파일들을 찾아가서 원인을 분석하고 즉시 수정안을 제시해줘.
2. **성능 체크**: 빌드가 성공하면, 결과물 중 사이즈가 비정상적으로 큰 청크(Chunk)가 있는지 확인하고 코드 분할(Code Splitting)이나 최적화가 필요한 부분을 제안해줘.
3. **환경 확인**: `.env` 설정이나 Node.js 버전 문제로 인한 빌드 실패 가능성도 함께 체크해줘.

명령어 실행 후 결과를 보고하고, 수정이 필요한 경우 내 승인을 받아 진행해줘.
